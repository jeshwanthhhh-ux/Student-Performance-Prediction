# Student Performance Prediction Using Machine Learning

## Project Overview

This project aims to predict students' final academic performance using Machine Learning techniques. The dataset contains demographic, social, and academic information about students. Through data preprocessing, exploratory data analysis (EDA), visualization, and predictive modeling, the project identifies key factors that influence student success.

## Objectives

- Analyze student performance data.
- Identify factors affecting academic results.
- Build a machine learning model to predict final grades.
- Evaluate model accuracy and performance.
- Generate insights that can help educational institutions support students.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- Google Colab
- GitHub

## Machine Learning Algorithm

- Random Forest Regressor

## Project Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Model Training
6. Model Evaluation
7. Prediction and Insights

## Results

The model successfully predicts student final grades and identifies important factors such as previous grades, study time, failures, and absences that significantly influence academic performance.
